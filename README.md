### Quick Download & Use (Recommended)
1. Click the green **Code** button at the top  
2. Select **Download ZIP**  
3. Extract the zip to any folder  
4. Open Zen Studio  
5. Browse to the /scripts/ folder inside the extracted directory  
6. Open any .gpc file → Compile (F7) → Connect Zen via USB → Drag to slot  
7. Save & test in 2K26 – start breaking ankles

All scripts are plain text – edit, combine, tweak as needed.
